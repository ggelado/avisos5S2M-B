                            <p>Hola a todos,</p>
<p>os recordamos que el examen de evaluación ordinaria de la asignatura es este <strong>Martes 2 de junio, a las 15:00 horas</strong>.</p>
<ul>
<li>Este examen lo deben realizar <strong>todos los alumnos</strong>.&nbsp;</li>
<li>Además, <em>aquellos que no hayáis entregado las prácticas en el primer plazo o no las tengáis aprobadas</em><strong> </strong>(media de las dos prácticas &gt;= 5) debéis además entregar, hasta el mismo día del examen, las prácticas que os falten o que necesitéis mejorar para aprobar las prácticas. Hay entregas específicas en Deliverit, marcadas "segundo plazo".&nbsp;</li>
</ul>
<p><strong>Información adicional sobre el examen:</strong></p>
<ul>
<li>Como orientación, el examen durará entre 35 y 50 minutos.</li>
<li>Debéis traer al examen el <strong>carné de estudiante o DNI </strong>y dejarlo a la vista encima de la mesa.</li>
<li>Debéis traer también un <strong>bolígrafo negro o azul</strong> para rellenar la hoja de respuestas.</li>
<li>Será un cuestionario en papel de 14 preguntas de opción múltiple con una respuesta correcta:&nbsp;<br>&nbsp; &nbsp; &nbsp; 5 preguntas de Programación Lógica Pura<br>&nbsp; &nbsp; &nbsp; 5 preguntas de ISO-Prolog<br>&nbsp; &nbsp; &nbsp; 2 preguntas de la Práctica 1<br>&nbsp; &nbsp; &nbsp; 2 preguntas de la Práctica 2<br>No entra la parte de programación lógica con restricciones (constraints)</li>
<li>Las contestaciones incorrectas no restarán.</li>
<li>Entregaremos a cada alumno un documento con las preguntas y una hoja de respuestas. &nbsp;</li>
<li>Cada alumno recibirá un examen diferente.</li>
</ul>
<p><strong>MUY IMPORTANTE: Debéis</strong>&nbsp;<strong>conocer vuestro Id de usuario Moodle</strong>, ya que es uno de los datos a rellenar tanto en el documento con las preguntas como en la hoja de respuestas.&nbsp;Este Id se puede consultar en el perfil del usuario en Moodle.</p>
<ul>
<li>En la hoja de preguntas se debe escribir el nombre, el Id de usuario Moodle, la asignatura y firmar</li>
<li>En la hoja de respuestas, se debe escribir el nombre, los apellidos, la firma y el Id de usuario Moodle.</li>
<li>El Id de usuario Moodle debe escribirse primero en cifras, alineado a la derecha con tantos ceros a la izquierda como sea necesario para completar las seis cifras y también como marcas debajo.</li>
<li>Las respuestas se deben contestar mediante marcas en la hoja de respuestas siguiendo las instrucciones incluidas en la propia hoja. &nbsp; &nbsp;</li>
</ul>
<p>Ver las imágenes adjuntas.&nbsp;</p>
<ul>
<li>Las preguntas pueden ser tipo "cuál es el mejor programa de los siguientes para implementar ...", "cuál sería el mejor caso base para un programa que hace ...", "dado el siguiente programa y consulta, cuál de estas respuestas es la correcta", "cuál sería la consulta apropiada para obtener estas respuestas", etc. Básicamente debéis entender y saber hacer programas similares a los predicados de las prácticas o los ejemplos de las transparencias de clase (programación lógica pura e ISO Prolog) y de los ficheros .pl que hay en Moodle que van con ellas.</li>
<li>En Moodle hay un ejemplo de examen. Contiene preguntas ejemplo sobre el material del curso, no de la parte de las prácticas.</li>
</ul>
<p>Buena suerte con el examen. Nos vemos el martes.&nbsp;</p>
<p>Un saludo, &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<br>El equipo docente de la asignatura &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<br>Programación Declarativa: Lógica y Restricciones&nbsp;</p>
